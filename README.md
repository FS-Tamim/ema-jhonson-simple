# ema-jhonson-simple

**Live website:** https://ema-jhonson-simple-41f5c.web.app/

**Features**
1.This is an e-commerce website where a customer can see all products easily.

2.Customer can add to cart .

3.Customer can see the cart and total price.

4.To check out customer will be redirect to login page.

5.After logging in can can check out.

6.Customer can write shipment details and complete the order easily.

7.There is a firebase based login system.

8.Customer can login with google account or previously registered email and password.

9 Customer can logout from the website.



**Technologies:** ReactJS, react router, NodeJS, MongoDB, HTML5, CSS3, Bootstrap4, MaterialUI

**Hosting & Authentication website:** Heroku, Firebase
